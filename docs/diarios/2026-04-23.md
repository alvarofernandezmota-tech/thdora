  ```
  ⚠️ Esta acción no se puede deshacer.
  ```

  igualmente la confirmación (el borrado puede continuar).

## 🔜 Siguiente hito — Tarea 1.2: Huecos disponibles al editar hora

### Objetivo
Cuando el usuario pulsa ✏️ → **Hora** para editar una cita, en lugar de pedirle
`HH:MM` a ciegas, mostrarle los **slots libres del día como botones**.

### Comportamiento esperado
```
Usuario pulsa ✏️ → "⏰ Hora"

Bot responde:
  🗓️ Huecos disponibles el lun 27 abr:

  🌅 Mañana       🌆 Tarde        🌙 Noche
  [08:00] [09:00] [14:00] [15:00] [22:00]
  [10:00] [11:00] [16:00] [18:00]
             [✏️ Escribir hora exacta]

  (grises = ocupados, verdes = libres)
```

### Implementación

**Archivos a tocar:**
- `src/bot/keyboards.py` → nueva función `_kb_slots_libres(apts, franja_key)`
- `src/bot/handlers/citas.py` → en `cb_apt_edit_field_chosen`, cuando `field == "time"`,
  en vez de pedir texto directo, cargar citas del día y mostrar los huecos

**Nueva función en keyboards.py:**
```python
def _kb_slots_libres(apts: list, franja_key: str) -> InlineKeyboardMarkup:
    """
    Muestra horas libres de una franja como botones.
    Los slots ocupados se omiten (no se muestran como deshabilitados,
    simplemente no aparecen — Telegram no soporta botones deshabilitados).
    Siempre incluye "✏️ Escribir exacta" como última opción.
    """
    horas_map = {
        "manana": list(range(6, 14)),
        "tarde":  list(range(14, 22)),
        "noche":  list(range(22, 24)) + list(range(0, 6)),
    }
    ocupadas = {a["time"][:2] for a in apts}  # set de horas en punto ocupadas
    horas    = horas_map.get(franja_key, list(range(6, 22)))
    libres   = [h for h in horas if f"{h:02d}" not in ocupadas]

    rows, row = [], []
    for h in libres:
        row.append(InlineKeyboardButton(f"{h:02d}:00", callback_data=f"ehueco_{h:02d}:00"))
        if len(row) == 4:
            rows.append(row)
            row = []
    if row:
        rows.append(row)

    if not libres:
        rows.append([InlineKeyboardButton("😅 No hay huecos en esta franja", callback_data="ehueco_none")])

    rows.append([InlineKeyboardButton("✏️ Escribir hora exacta", callback_data="ehueco_exacta")])
    rows.append([InlineKeyboardButton("← Cancelar",             callback_data="cancel_action")])
    return InlineKeyboardMarkup(rows)
```

