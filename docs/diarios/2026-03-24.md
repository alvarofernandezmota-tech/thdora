## 🌙 Sesión noche (22:00 — 22:55 CET)

### Objetivo
Subir el código generado por Claude (tareas 1-4) al repo y resolver el entorno WSL2.

### Problemas resueltos

| Problema | Solución |
|----------|----------|
| `wsl --shutdown` no funciona en WSL | Comando solo válido en PowerShell Windows, no en bash WSL |
| `unzip` no instalado | `sudo apt install unzip -y` |
| ZIP no encontrado en `/mnt/c/Users/Alvaro/` | Usuario Windows real es `alvar` (sin la o): `/mnt/c/Users/alvar/` |
| `git commit` falla: empty ident | `git config --global user.email` y `user.name` no configurados |
| `git push` rechazado (fetch first) | Repo tenía commits nuevos → `git pull origin main --rebase` |
| Password authentication not supported | GitHub ya no acepta password → usar Personal Access Token (PAT) |
| PAT pedido en cada push | `git config --global credential.helper store` → guardado permanente |
