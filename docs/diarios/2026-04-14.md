## 🚀 Cómo lanzar el bot ahora para probar

```bash

# 2. Activar entorno virtual (si no está activo)

# 3. Asegurarse de que .env tiene las claves necesarias

# GROQ_API_KEY=gsk_...
```

## 💡 Decisiones tomadas hoy

| Decisión | Razón |
|----------|---------|
| NO integrar thea-ia con THDORA | Complejidad injustificada para uso personal; Groq ya hace lo mismo mejor |
| PicklePersistence en lugar de Redis | Más simple, sin dependencias extra, suficiente para 1 usuario |
| Personalidad en `_CHAT_SYSTEM` hardcodeada | No necesita panel de config; se edita directo en 10 segundos |
| Fix 00:00 en nlp.py, no en groq_router | El router debe devolver lo que Groq da; la lógica de negocio va en el handler |
| `store_bot_data=False` en PicklePersistence | El `bot_data` es vacío, no tiene sentido persistirlo y aumentar el pkl |
